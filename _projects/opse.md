OPSE is a companion app for solo tabletop roleplaying — built to handle the
bookkeeping (dice, oracles, tables) so the story stays the focus rather than
the spreadsheet.

## Why it exists

Solo RPGs lean heavily on random oracles and tables to generate narrative
beats. Flipping between PDFs and dice rolls breaks immersion quickly, so
OPSE consolidates that into a single, fast interface.

## Notable bits

- Built with Jetpack Compose and an MVVM architecture, using `StateFlow`
  for reactive state across screens.
- Designed for offline-first use — no network dependency mid-session.
